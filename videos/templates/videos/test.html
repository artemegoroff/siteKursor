Условный оператор if позволяет выполнить определенный набор инструкций в зависимости от некоторого условия. Использовать этот оператор можно в нескольких вариантах. В этом видео разберем только 2 варианта.

1 Конструкция if

В простейшем варианте использование if выглядит так:

if условие:
    блок команд 1
блок команд 2

После условного оператора записывает условие. Если данное условие будет истинно, то выполняется блок команд 1. После условия нужно поставить двоеточие “:”.
Вне зависимости от условия блок команд 2 будет выполняться в любом случае.

Примеры использования
if a==777:
    print("Этот принт сработает, если а будет равно 777.")
    print("И так же этот принт.")
    print("И этот.")

print("А это сообщение выведется в любом случае")

Еще пример 

money = 100
ticket = 90
if money > ticket:
    print("УРАААА!!!")
    print("Я иду в кино!")
print("Пора идти домой")

Что касается отступов. Внутри блока инструкции должны находиться на одинаковом уровне отступов. В противном случае будет ошибка.

if 5>1:
  print("Тут оступ в 2 пробела.")
    print("А здесь в четыре. Будет ошибка.")

a=int(input())
b=int(input())
if b>a:
    a,b = b,a
print(a,b)

2 Конструкция if-else

if условие:
    блок команд 1
else:
    блок команд 2
блок команд 3

Общая форма записи условного оператора if выглядит следующим образом:

if условие1:  # Инструкция if с условным выражением test1
  блок1      # Ассоциированный блок
elif условие2:  # Необязательные части elif
  блок2
else:    # Необязательный блок else
  блок3
